# Next.js
## Beschrijving:

Het project is gebouwd met React, de populaire bibliotheek voor interactieve en herbruikbare UI-componenten, en maakt gebruik van Next.js, een framework bovenop React dat routing, server-side rendering en andere geavanceerde functies biedt om de website snel en schaalbaar te maken.

### Wanneer gebruik je Next.js?
- Als je React wilt gebruiken.
- Als je dynamische routes, API-routes of server-side rendering nodig hebt.
- Als je een interactieve webapplicatie wilt bouwen.

## Installatie

### Clone de repository
git clone https://github.com/Abeeryu/React-Next.js.git

### Ga naar de projectmap
cd React-Next.js

### Installeer dependencies
npm install

### Start de development server
npm run dev

## Technologieën

Next.js – React framework voor server-side rendering en routing

React – Voor interactieve UI-componenten

Tailwind CSS – Voor snelle en moderne styling

## Structuur van het project

app/ – Bevat de pagina’s en layout van de applicatie

globals.css – Globale CSS en Tailwind imports

package.json – Dependencies en scripts

.next/ – Build map (wordt automatisch gegenereerd)

## Gebruik

- Voeg pagina’s toe in app/ met een .tsx bestand.

- Pas layout aan in app/layout.tsx.

- Styling kan eenvoudig met Tailwind CSS-klassen.
